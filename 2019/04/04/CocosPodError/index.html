执行 pod setup 很慢很慢的解决方案

前期准备(安装过可以省略)：1.查看ruby 2.更新 gem 3.安装cocoapods。

  1. pod repo remove master

  2. pod repo add master https://gitcafe.com/akuandev/Specs.git

这一步把https改成git 速度上会快一点

  2. pod repo update
